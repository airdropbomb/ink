## Setup & Configure BOT

### Linux

1. clone project repo
   ```
   git clone https://github.com/airdropbomb/ink.git
   cd ink
   ```
2. run
   ```
   npm install
   npm run setup
   ```
3. Configure your accounts
   ```
   nano privateKeys.txt
   ```
4. to start the app run
   ```
   npm run start
   ```

### Windows

1. Open your `Command Prompt` or `Power Shell`.
2. Clone project repo
   ```
   git clone https://github.com/airdropbomb/ink.git
   ```
   and cd to project dir
   ```
   cd ink
   ```
3. Run
   ```
   npm install
   npm run setup
   ```
4. Navigate to `ink` directory.
5. Navigate to root and configure `privateKeys.txt`.
6. Back to `ink` directory.
7. To start the app open your `Command Prompt` or `Power Shell` again and run
   ```
   npm run start
   ```

## Update Bot

To update bot follow this step :

1. run
   ```
   git pull
   ```
   or
   ```
   git pull --rebase
   ```
   if error run
   ```
   git stash && git pull
   ```
2. run
   ```
   npm update
   ```
3. start the bot
